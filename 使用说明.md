# 📘 刷题管理系统 - 使用说明

## 📌 系统简介

这是一个智能刷题管理系统，帮助你自动整理 C++ 刷题文件、统计学习进度、AI 分析薄弱环节，并生成每日训练计划。

**核心功能：**
- 📁 自动扫描 `.cpp` 文件，识别题目类型和平台
- 📊 Web 可视化面板，展示刷题统计
- 🤖 AI 学习建议和智能题目推荐
- 📅 每日训练计划生成
- 🌐 多平台进度同步（洛谷、Codeforces、AtCoder、牛客）

## 🚀 快速启动

### 方法一：使用批处理文件（推荐）

1. 双击打开 `start.bat` 文件
2. 等待服务启动（看到类似 `Running on http://127.0.0.1:5000` 的提示）
3. 在浏览器中访问：**http://localhost:5000**

### 方法二：手动启动

1. 打开命令提示符（CMD）或 PowerShell
2. 进入项目目录：
   ```cmd
   cd c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统
   ```
3. 激活虚拟环境：
   ```cmd
   venv\Scripts\activate
   ```
4. 启动 Web 服务：
   ```cmd
   python web\app.py
   ```
5. 在浏览器中访问：**http://localhost:5000**

## 🎯 首次使用配置

### 1. 配置刷题文件夹路径

1. 访问 http://localhost:5000/settings
2. 在"目标文件夹"中填入你的刷题文件路径，例如：
   ```
   E:\大学\编程\Save-point\日常刷题
   ```
3. 点击"保存配置"

### 2. 扫描现有题目

1. 在设置页面点击"📂 立即扫描文件"
2. 系统会自动扫描文件夹中的所有 `.cpp` 文件
3. 扫描完成后，题目会自动分类并存入数据库

### 3. 配置平台账号（可选）

如果你想同步平台进度，可以配置各平台账号：

1. 在设置页面填写平台账号信息：
   - **洛谷用户名**：例如 `lt_wxy`
   - **Codeforces Handle**：例如 `ltwxy`
   - **AtCoder 用户名**：选填
   - **牛客昵称**：选填

2. 填写完成后点击"保存配置"

3. 点击"🔄 获取平台进度"来同步你的刷题数据

### 4. 配置 DeepSeek API（可选）

AI 功能需要 DeepSeek API Key：

1. 访问 [DeepSeek 官网](https://platform.deepseek.com/) 注册并获取 API Key
2. 在设置页面填入你的 API Key（格式：`sk-xxx`）
3. 点击"保存配置"

**注意**：API Key 仅保存在本地 `config.json` 文件中，不会上传到任何服务器。

## 🔄 每日刷新机制

### 自动刷新

系统会在以下情况下自动刷新数据：

1. **文件监听**：当你修改或新增 `.cpp` 文件时，系统会自动扫描并更新数据库
2. **AI 建议缓存**：AI 学习建议会缓存 30 分钟，避免频繁调用 API
3. **每日计划**：每天第一次访问计划页面时，会检查是否需要生成新计划

### 手动刷新

如果需要立即刷新数据：

1. **刷新统计数据**：
   - 在主页按 F5 刷新页面
   - 或点击"📊 刷新统计"按钮（如果有的话）

2. **刷新 AI 建议**：
   - 删除 `data\ai_advice_cache.json` 文件
   - 在主页按 F5 刷新页面

3. **刷新平台进度**：
   - 访问设置页面
   - 点击"🔄 获取平台进度"

4. **重新生成每日计划**：
   - 访问计划页面
   - 点击"🎲 重新生成计划"

## ❓ 常见问题

### Q1: 启动后无法访问 http://localhost:5000？

**可能原因：**
- 端口 5000 被其他程序占用
- 防火墙阻止了访问

**解决方案：**
1. 修改 `web\app.py` 的最后一行，将端口改为其他数字（如 5001）：
   ```python
   app.run(debug=True, port=5001)
   ```
2. 或者关闭占用 5000 端口的程序

### Q2: 文件扫描不到题目？

**可能原因：**
- 文件名不符合命名规范
- 文件不在配置的目标文件夹中
- 文件扩展名不是 `.cpp`

**解决方案：**
1. 检查文件名是否符合命名规范
2. 在设置页面确认目标文件夹路径正确
3. 点击"📂 立即扫描文件"手动触发扫描

### Q3: AI 建议不显示？

**可能原因：**
- 没有配置 DeepSeek API Key
- API Key 无效
- 网络问题

**解决方案：**
1. 确认已在设置页面填写 DeepSeek API Key
2. 访问 [DeepSeek 官网](https://platform.deepseek.com/) 检查 API Key 是否有效
3. 检查网络连接是否正常

### Q4: 平台进度同步失败？

**可能原因：**
- 账号信息填写错误
- 平台限制了访问
- 网络问题

**解决方案：**
1. 确认平台账号填写正确
2. 尝试直接访问平台网站，确认账号可正常访问
3. 检查网络连接是否正常
4. 某些平台可能需要登录，可以尝试在设置页面填写 Cookie

### Q5: 每日计划生成的题目太难/太简单？

**可能原因：**
- 系统根据你的掌握度调整难度，可能需要时间学习你的水平

**解决方案：**
1. 完成任务时，可以在"备注"中说明题目难度
2. 跳过不合适的题目
3. 多完成一些题目后，系统会逐渐了解你的水平，推荐的题目会更加精准

### Q6: 如何停止服务？

**方法一**：在命令窗口中按 `Ctrl + C`

**方法二**：直接关闭命令窗口

**方法三**：在任务管理器中结束 `python.exe` 进程
