为什么要遵循命名规范？

正确的文件命名可以让系统自动识别题目来源平台、题号、题目名称和使用的算法，从而生成准确的题目链接和分类统计。

🏛️ 洛谷 (Luogu)

题号格式 示例 生成的链接
B + 数字 入门 B3626_跳跃机器人.cpp luogu.com.cn/problem/B3626
P + 数字 普及/提高 P1314_聪明的质检员.cpp luogu.com.cn/problem/P1314
T + 数字 团队题 T123456_团队题目.cpp luogu.com.cn/problem/T123456
U + 数字 UVA U12345_题目名.cpp luogu.com.cn/problem/U12345
AT + 编号 AtCoder AT_abc123_a.cpp luogu.com.cn/problem/AT_abc123_a
CF + 数字 Codeforces CF1234A_题目名.cpp luogu.com.cn/problem/CF1234A

推荐命名格式

题号_题目名称(算法标签).cpp

示例： B2005_字符三角形.cpp
P1219_八皇后(dfs).cpp
P4779_单源最短路径(dijkstra).cpp

🔵 Codeforces

题号格式 示例 生成的链接
CF + 比赛号 + 题号 CF1234A_题目名.cpp codeforces.com/problemset/problem/1234/A
比赛号 + 题号（无前缀） 1234A_题目名.cpp codeforces.com/problemset/problem/1234/A
提示： Codeforces 题号通常是「比赛编号 + 题目字母」，如 1822A、1699C 等。

🇺🇸 USACO

题号格式 示例
USACO + 年份 + 月份 + 题目 USACO_2023_Dec_B1.cpp
直接使用题目名 USACO_milk_pails.cpp

🇨🇳 NOIP / CSP / 省选

题目类型 示例

CSP-J/S CSPJ2022_乘方.cpp

分类 标签示例

DP dp 背包 区间dp 状压dp 树形dp

⚠️ 注意事项

避免使用的命名：
  • 纯日期格式：2026_2_23.cpp（系统会跳过识别）
  • 无意义的名称：test.cpp、1.cpp
  • 中文特殊符号：题目【1】.cpp（可能导致编码问题）
最佳实践：
  • 题号放在最前面，便于排序和识别
  • 用下划线 _ 分隔题号和题目名
  • 算法标签放在括号内，系统会自动提取
  • 题目名可以省略，系统会从代码中提取
