# Windows 开机自启设置教程

## 方法一：快捷方式到启动文件夹（推荐，最简单）

### 步骤

1. **创建快捷方式**
   - 打开文件管理器，进入项目文件夹：
     ```
     c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统
     ```
   - 找到 `start.py` 文件
   - 右键点击 `start.py` → **发送到** → **桌面快捷方式**

2. **移动到启动文件夹**
   - 打开运行窗口：按 `Win + R`
   - 输入以下路径并回车：
     ```
     shell:startup
     ```
   - 这会打开启动文件夹
   - 把桌面上刚创建的快捷方式拖到这个文件夹里

3. **完成**
   - 重启电脑或注销后重新登录
   - 刷题管理系统会自动启动
   - 等待10秒后打开浏览器访问 http://localhost:5000

## 方法二：创建批处理文件（更稳定）

### 步骤

1. **创建批处理文件**
   - 在项目文件夹 `c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统` 下
   - 新建一个文本文件，命名为 `start_system.bat`
   - 用记事本打开，粘贴以下内容：
     ```batch
     @echo off
     cd /d "c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统"
     python start.py
     ```

2. **创建快捷方式**
   - 右键点击 `start_system.bat` → **发送到** → **桌面快捷方式**

3. **移动到启动文件夹**
   - 按 `Win + R`，输入 `shell:startup` 并回车
   - 把快捷方式拖到启动文件夹

4. **完成**
   - 重启或注销后自动启动

## 方法三：任务计划程序（最灵活，可设置延迟启动）

### 步骤

1. **打开任务计划程序**
   - 按 `Win + S` 搜索「任务计划程序」
   - 打开应用

2. **创建基本任务**
   - 右侧点击 **创建基本任务**
   - 名称输入：`刷题管理系统`
   - 点击 **下一步**

3. **设置触发器**
   - 选择 **计算机启动时**
   - 点击 **下一步**

4. **设置操作**
   - 选择 **启动程序**
   - 点击 **下一步**

5. **配置程序**
   - 程序或脚本：输入 `python.exe` 的完整路径
     - 如果使用系统 Python：`C:\Users\34694\AppData\Local\Microsoft\WindowsApps\python.exe`
   - 添加参数（可选）：
     ```
     "c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统\start.py"
     ```
   - 起始于（可选）：
     ```
     c:\Users\34694\WorkBuddy\20260331104533\刷题管理系统
     ```
   - 点击 **下一步**

6. **完成**
   - 勾选 **打开此任务属性的对话框**
   - 点击 **完成**

7. **配置延迟启动（推荐）**
   - 在属性窗口，切换到 **触发器** 标签
   - 双击触发器，点击 **高级设置**
   - 勾选 **延迟任务**，设置为 **30秒** 或 **1分钟**
   - 点击确定，再点击确定

8. **完成**
   - 重启电脑，系统启动后会延迟30秒自动运行刷题管理系统

## 测试是否设置成功

1. **立即测试（无需重启）**
   - 在任务计划程序中，找到刚创建的任务
   - 右键点击 → **运行**
   - 等待几秒，打开浏览器访问 http://localhost:5000
   - 如果能打开，说明设置成功

2. **重启测试**
   - 重启电脑
   - 登录后等待1-2分钟
   - 打开浏览器访问 http://localhost:5000

## 停止自动启动

如果以后不想开机自动启动：

### 方法一：删除启动文件夹中的快捷方式
1. 按 `Win + R`，输入 `shell:startup`
2. 删除 `start.py` 的快捷方式

### 方法二：禁用任务计划程序
1. 打开任务计划程序
2. 找到 `刷题管理系统` 任务
3. 右键点击 → **禁用**
   或右键点击 → **删除**

## 常见问题

### Q1: 启动后看不到窗口？

**A**: 系统是在后台运行的。打开浏览器访问 http://localhost:5000 就能看到界面。

### Q2: 提示找不到 python？

**A**: 确认 Python 已安装。可以在命令行输入 `python --version` 检查。

### Q3: 多次启动导致端口冲突？

**A**: 如果电脑重启，旧进程会自动结束。如果是手动启动多次，可能需要手动结束进程。按 `Win + X` → **任务管理器**，找到 python 进程并结束。

### Q4: 如何查看是否启动成功？

**A**: 打开浏览器访问 http://localhost:5000，如果能看到刷题管理系统界面，说明启动成功。
