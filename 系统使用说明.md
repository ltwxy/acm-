# 刷题管理系统 - 使用说明

## 📋 系统概述

这是一个智能刷题管理系统，自动识别本地刷题文件，整合多平台（洛谷、Codeforces、AtCoder、牛客网）数据，通过 AI 分析给出个性化训练建议。

### 核心功能

1. **本地刷题自动扫描** - 监控指定文件夹，自动识别 `.cpp` 文件并提取题目信息
2. **多平台进度同步** - 绑定竞赛平台账号，获取已解题列表
3. **AI 智能分析** - 基于 DeepSeek API 生成 SWOT 分析、弱点分析、学习建议
4. **每日训练计划** - 根据你的刷题数据和薄弱环节，智能推荐每日题目
5. **综合统计面板** - 可视化展示知识点分布、掌握度、难度分布等
6. **AI 对话助手** - 支持基于上下文的智能题目推荐和学习指导

## ⚙️ 系统工作原理

### 1. 文件监控与自动扫描

**监控机制**：
- 使用 `watchdog` 库实时监控刷题文件夹
- 当检测到新的 `.cpp` 文件时自动解析文件名

**支持的文件名格式**：

洛谷：
```
B3644.cpp           # 简单格式：题号
P1000.cpp           # P 题
B1000.cpp           # B 题
T1000.cpp           # T 题
U1000.cpp           # U 题
luogu_P1000.cpp     # 带前缀
```

Codeforces：
```
2184D.cpp           # contestID + index
codeforces_2184D.cpp # 带前缀
cf_2184D.cpp        # 简写前缀
```

USACO：
```
usaco_barn1.cpp     # usaco 题目名（小写+下划线）
```

NOIP：
```
noip_greeting.cpp   # noip 题目名
```

通用：
```
# 任意包含题目的 .cpp 文件
```

**自动提取信息**：
- 算法类型（从文件名关键词匹配：DP、贪心、图论、搜索、数学等）
- 题目难度（洛谷 1-7，CF rating 映射到 1-10）
- 提交时间

### 2. 多平台数据爬取

**支持的平台**：
- **洛谷** - 通过题目 ID 批量查询
- **Codeforces** - 通过用户名获取已提交题目
- **AtCoder** - 通过用户名获取 AC 题目
- **牛客网** - 通过 API 获取已解题

**爬取内容**：
- 题目基本信息（标题、难度、标签）
- 题目链接
- 已解题列表

**难度归一化**：
- 洛谷：1-7 直接使用
- Codeforces：800-1000 → 2, 1000-1200 → 3, ..., 2400+ → 10
- 牛客：简单 → 3, 中等 → 6, 困难 → 9

### 3. AI 智能分析

**SWOT 分析**：
- 分析你的强项和薄弱知识点
- 给出针对性的改进建议
- 生成详细的文字报告

**弱点分析**：
- 识别最薄弱的知识点（如：动态规划）
- 推荐强化训练题目

**每日计划生成**：
- 根据你的刷题数据和薄弱环节
- 推荐未做过的平台题目
- 智能生成每日训练计划

**AI 对话助手**：
- 支持多轮对话
- 基于你的刷题历史给出精准建议
- 可推荐具体题目并生成链接

### 4. 数据去重与合并

**去重规则**：
- 同一题目（平台 + ID）只计算一次
- 本地文件和平台数据合并统计
- 避免重复计数

**合并统计**：
- 总刷题数 = 本地题目 + 平台题目（去重）
- 知识点分布 = 所有题目的标签聚合
- 难度分布 = 所有题目的难度统计

## 📅 刷新机制

### 自动刷新

| 刷新内容 | 刷新时机 | 说明 |
|---------|---------|------|
| 本地文件扫描 | 实时 | 监控文件夹，新增文件自动识别 |
| 平台数据 | 手动触发 | 点击设置页面的「爬取平台题目」按钮 |
| AI 分析报告 | 手动触发 | 点击主页的「生成分析报告」按钮 |
| 每日训练计划 | 手动触发 | 点击每日计划页面的「重新生成」按钮 |
| AI 建议 | 30分钟缓存 | 同一分析结果缓存30分钟，超时自动重新生成 |

### 手动刷新方法

**刷新平台数据**：
1. 打开 http://localhost:5000/settings
2. 确认平台账号已配置
3. 点击「爬取平台题目」按钮
4. 等待爬取完成（约 30-60 秒）

**刷新每日计划**：
1. 打开 http://localhost:5000/plan
2. 点击「重新生成计划」按钮
3. 等待 AI 推荐完成（约 10-30 秒）

**刷新 AI 分析**：
1. 打开 http://localhost:5000/
2. 在统计面板中点击「生成分析报告」或「查看详细分析」
3. 等待 AI 分析完成（约 30-60 秒）

## ❓ 常见问题

### Q1: 每日计划推荐的题目链接打不开？

**A**: 确保已爬取平台数据。点击「重新生成计划」即可更新链接。

### Q2: 本地文件扫描不到？

**A**: 检查设置页面的文件夹路径是否正确，确认文件名符合命名规范。

### Q3: AI 分析报错？

**A**: 检查 `config.json` 中的 DeepSeek API Key 是否正确配置。

### Q4: 如何重置系统？

**A**:
1. 停止 Web 服务器
2. 删除 `data/problems.db` 数据库文件
3. 重新启动系统（会自动创建新数据库）

### Q5: 可以同时使用多个平台吗？

**A**: 可以！在设置页面配置所有需要的平台账号，系统会自动合并统计数据。
